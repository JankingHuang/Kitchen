# Kitchen

kitchen是个人毕业设计的移动监控系统，项目使用AndroidX命名空间的库。

## 项目使用的框架
- Okhttp
- - >https://square.github.io/okhttp/
- Gson
- - >https://github.com/google/gson
- Glide
- - >https://bumptech.github.io/glide/
- MPAndroidChart
- - > https://github.com/PhilJay/MPAndroidChart
